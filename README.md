# TrustLayer

> A trust layer that verifies AI-generated code against official documentation.

## Why?

AI is great at generating code.

Sometimes it's also great at confidently inventing commands, flags, and configurations that don't exist.

TrustLayer aims to help developers verify AI-generated output against trusted sources before executing it.

## Status

🚧 Early development

The project is currently in the research and validation phase.

After collecting feedback from Reddit and other developer communities, I'm working on the first MVP.
